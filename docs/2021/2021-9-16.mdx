# 我体外的现实终将南下

<div style={{ whiteSpace: "pre-line"}}>

混凝土是纳粹的象征
我走在暴力的边缘
阳光默许我这样
泥土僵硬，大道干涸
我，滚动的石子
也和你们一起赴宴

分一杯羹给我
让脊梁挺直
身体肮脏，臭不可闻

于顿悟中腐烂
一切的父亲都来自
浓黑的性，而非刺入

高楼仿若佛像
充斥着摇滚的喊叫
罪行不现，念念成空
不为谁的祈祷
我弯腰跌落跪倒
风中呢喃依旧

2021.9.16
