# 问李白

<div style={{ whiteSpace: "pre-line"}}>

今有一月，我有一酒
停杯一问，李白一答
不问酒，不问月
不问诗，不问人
但问我在何方

我已醉，舞影杀四方
影未饮，未与月醉
不敢动刀，遁地而逃

李白大笑三声
一笑我诺诺如影
二笑我穷酸无酒
三笑我有月不赏

那月为我大呼争辩
满面泣痕，斥骂李白
一千多年前，是谁狂歌高唱
古来圣贤皆寂寞，惟有饮者留其名
天子呼来不上船，自称臣是酒中仙

我已与月醉，李白与酒醉
月是酒的问，酒是月的答
仿佛如此了千年万年
他日不见李白与我
月啊，你与酒醉吧
你与月影醉吧

2021.4.22
