# 几十年的一个夜晚

<div style={{ whiteSpace: "pre-line"}}>
爱了几十年，也未曾得到回应
写了几十年，也不敢小心沉默
突然一阵心酸，未觉惭愧
醒来在黑夜中迷茫，也挺快活

几十年了，几十年的昨天还在说
几十年的夜晚该如何如何想你

泛黄的纸墨何尝不勾起你的柔情
泪水浸满了星辰，却溺死月光
也就会有那么一个长长的黄昏
我见你向我起来，带着远古的妩媚
不敢靠近，又不得不去谈吐
消磨了一个几十年前，一个几十年后
几十年前的等待，几十年后的永恒

走在两个几十年间的小巷里
渐渐平息掉无尽的躁动喧闹
一点一点的思念将青春缠绕
我的禅谜开始浮出水面
没有告别，如何证明曾经相遇过？

当你老去，彻悟已经如影随形
一切都倒映着滚烫的生活

2019.12.20
