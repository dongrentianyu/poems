# 昨日的疯狂，好笑的爱

<div style={{ whiteSpace: "pre-line"}}>
你跪在生命前，虔诚祈祷
从不在乎白昼的挽留
我将岁月踩在脚下
一直等待黑夜的忏悔

你有着春风拂面的温柔
我却看出万丈深渊的冷漠
你从不见我，只与我偶遇
那还是蓝色夜晚的回忆

一点点无情足以燃烧我的爱
用疯狂乞讨梦幻般的昨日
干涩的月影下放纵痛苦
疲倦缓缓散发出虚假，恶心

你还不明白吗？
爱是怀疑了的孤独

2019.12.4
