# 把人抛

<div style={{ whiteSpace: "pre-line"}}>

我醒来，哭泣
无言也呜咽
如此活生生地苟活

你大笑，该
就这样委命自流
以大丈夫之道
绝万圣之绝学

历史总在沉默中爆炸
灰烬自然不足为惜
所以与光同尘
和而为罪

越来越多的痒和痛
变得撕心裂肺
密密麻麻
涌入，抓挠
直到一个个个体胀裂
生产咧嘴的丧尸

他梦了醉了梦了
旁无一亲一友
劝与念也要几钱
若问身前功名何在
一席凉被卷

2022.3.27
