# 玫瑰

<div style={{ whiteSpace: "pre-line"}}>

夜里，玫瑰轻声对我说
别把生活当成爱情的筹码
那不值得，说完笑了笑
一瓣一瓣地在眼前凋零飘落
我猛然惊醒，后背直冒冷汗

她病酒好多年了，还在喝
可你只在一旁默默抽着烟
她说了吵了闹了哭了
我还是守着默剧，等待落幕

一些欲念，早就收刀入鞘
连日常都需要逻辑来翻译
生活渐渐在石化中透明

早年野合下的玫瑰
已经吮尽了你我的爱
也就无须再亲吻再抚摸
也就有股莫名的疼痛
在恍惚之间刺向了虚空

2020.11.24
