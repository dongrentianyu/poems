# 最初的爱情，最后的仪式

<div style={{ whiteSpace: "pre-line"}}>

> 你终于闪耀了吗？
> 我旅途的终点。
> ————梵乐希

你还是在我的身体里
像鸟群一样迁徙
如此温柔的暴动

知你不来，我忍住不哭
再来一次如恋的黄昏
缓缓渗入那些期盼
静静酿出夜酒的香浓
一醉再醉，若醒复醉

多少思念滂沱的夜里
我只跟踪一次回忆
为什么你那么单薄

你己在我的诗里
写上一句，又一句

如此暴动的温柔
你总是在我的身体里
像鸟群一样迁徙

2020.8.20
