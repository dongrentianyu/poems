# 大地上的神谕

<div style={{ whiteSpace: "pre-line"}}>

> ——死去的人在风中飘荡
> 正如我们在时间奔走
> ————梁晓明《开篇之最初》

把太阳的眼睛带回故乡
让死去的血与泪从中喷涌
以后便是开篇里的最初了

夜空上的蔚蓝色弹奏着
时间在我们体内的哀鸣
成了重叠与挤压的统一
不幸他们所有的裂隙
都被偌大的激情所堵塞

言辞如砖瓦在嘴上筑起长城
他们才是风沙世代思念的老父亲
但摩西早己坦然剥开我的灵魂
像一片海行走在荒漠上

我隐约听见绿草在与月光生长
投向历史的目光也收回到前方
也许人类再一次由童年改写
闪电轰击了先知们的痛

预言因其实现而陷入灭亡
来来回回的水滴开始繁衍后代
直到天穹上的神谕降临在大地上

2020.7.7
