# 午夜当归

<div style={{ whiteSpace: "pre-line"}}>

黑夜里飞驰的火车
像生活一样
来不及欣赏那些
一晃而过的光点

诗人曾说过：
那片故土
只因我的到来
才真正有了
对远方的悸动

我也一样
每当黑夜横亘眼前
瞬间心如死灰
抽刀直刺
鏖战以不眠

火车呼呼前行
呜咽着载我逃离
可我知道
终有一站下车

把黑夜的遗腹子
诞在亮光的原野上
铁轨不再神秘了
孩子们依旧
快乐地随风起舞

2020.12.18
