# 我是大海上的森林

<div style={{ whiteSpace: "pre-line"}}>
我怀着一个海洋的梦
缓缓爬到你的脚边
你用激荡的波涛告诉我
自由是何等圣洁！何等高贵！

苦笑一声，斩断杂念
生长不过是死亡的偷袭
触摸被打湿的过往
那一天，我们都很美好

锋利的寒风将我苔伤
散去形如枯槁的躯体
海底的深渊逐渐吞噬
我若有若无的悲伤

啊！困倦的春天终于来了
炸出软绵绵慵懒的光芒
一片落叶一生只陨落一次
只为与心中的浪花融合
是向大海倾诉爱意，还是播种？

你有没有一片森林的梦呢？
扭曲粗糙腐朽痛苦的树枝
会不会在梦里将你惊醒
也许你不是我的森林

2020.1.12
