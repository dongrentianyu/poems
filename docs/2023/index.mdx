# 满地的虫和它们的嘉年华

<div style={{ whiteSpace: "pre-line"}}>

那是一个嘉年华
屹立着观光塔
假山镂空，姿态雄伟
水流清澈，残荷泛臭

锁链把我们选择的时间
关在了保安大叔的惊讶中
嘉年华早就过去了
你们还来干什么？

我们不解也不思地走进眼前
一切都在阳光下破败了
准备拍照的阴影里
墨镜带好，上帝要发笑了

其余的也没稀奇什么
只是树走成了路，路没了树
犹如超现实的梦境
满地的虫爬得大道满地都是
它们熙熙攘攘狂欢至死

生命可畏，岁月无惧
旁边的污水处理厂早已漠视
轰吟运转的机器与狂奔不暇的我们
陷入同谋，在逃离中扼杀
死亡嘉年华正式开幕

2023-9-13
