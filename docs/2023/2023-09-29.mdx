# 电子厂的年轻人们

<div style={{ whiteSpace: "pre-line"}}>

年轻人已有复数
还要加个们字
我也是他们的一员

面貌各异，习惯沉默
再没有了学校班里的燥动
毋须多言，他们也有他们的安排
年轻的命运也要在他们血液里沸腾

上车越多，生活的思念越多
把被直视的阳光拖运到另一世界
去生长，去膨胀，去销毁

智能产业制造园
像宇宙一样无声涰泣着
那是长夜难明的未来
那弥散了一路的桂花香
成为比梦更现实的天堂了

2023-09-29
